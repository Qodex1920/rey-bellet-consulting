Exemples de Cartes de Services

Cette page démontre les différentes variantes de cartes de services et comment les utiliser dans votre projet.

Cartes pour Entreprises

Design élégant avec fond sombre, icônes et liste de fonctionnalités pour mettre en valeur vos services aux entreprises.

Code d'exemple

// Création d'une carte enterprise
import { createServiceCard, serviceIcons } from '../components/common/ServiceCard.js';

const enterpriseCard = createServiceCard({
  title: 'Conseil en gestion du changement',
  description: 'Accompagnement des organisations dans leurs transformations stratégiques',
  variant: 'enterprise',
  features: [
    'Analyse des besoins organisationnels',
    'Planification stratégique',
    'Accompagnement des équipes'
  ],
  icon: serviceIcons.consulting,
  ctaText: 'Demander un devis',
  ctaUrl: '/contact.html'
});

Cartes pour Particuliers

Style premium avec badge optionnel, tags et effets de survol raffinés, idéal pour présenter des services aux particuliers.

Code d'exemple

// Création d'une carte personal
import { createServiceCard } from '../components/common/ServiceCard.js';

const personalCard = createServiceCard({
  title: 'Coaching personnel',
  description: 'Développez votre potentiel et atteignez vos objectifs personnels à travers un accompagnement sur mesure',
  variant: 'personal',
  tags: ['Développement', 'Coaching', 'Bien-être'],
  isPremium: true,
  ctaText: 'Prendre rendez-vous',
  ctaUrl: '/contact.html'
});

Cartes Simples

Version légère avec design épuré, parfait pour des présentations plus sobres ou dans des sections secondaires.

Code d'exemple

// Création d'une carte simple
import { createServiceCard, serviceIcons } from '../components/common/ServiceCard.js';

const simpleCard = createServiceCard({
  title: 'Formation en leadership',
  description: 'Acquérez les compétences essentielles pour diriger efficacement',
  variant: 'simple',
  icon: serviceIcons.training,
  ctaText: 'En savoir plus',
  ctaUrl: '/services.html#formation'
});

Utilisation programmatique

Vous pouvez également ajouter des cartes de services à des conteneurs existants par programmation.

Container cible

La carte sera injectée ici

Code d'exemple

// Ajout d'une carte à un conteneur existant
import { addServiceCardTo, serviceIcons } from '../components/common/ServiceCard.js';

addServiceCardTo('#programmable-container', {
  title: 'Accompagnement du changement',
  description: 'Solutions sur mesure pour faciliter les transitions organisationnelles',
  variant: 'enterprise',
  features: [
    'Diagnostic organisationnel',
    'Plan de transformation',
    'Ateliers participatifs'
  ],
  icon: serviceIcons.custom
});
