Exemples de Boutons

Cette page montre comment utiliser le composant Button.js de différentes manières dans votre projet.

Utilisation via attributs data

Les boutons peuvent être créés automatiquement en utilisant des attributs data-* sur un élément HTML.

Bouton primaire

<div 
  data-button="primary"
  data-text="Bouton primaire"
  data-url="#"
></div>

Bouton secondaire

<div 
  data-button="secondary"
  data-text="Bouton secondaire"
  data-url="#"
></div>

Bouton accent

<div 
  data-button="accent"
  data-text="Bouton accent"
  data-url="#"
></div>

Bouton avec icône

<div 
  data-button="primary"
  data-text="Découvrir"
  data-icon="arrowRight"
  data-url="#"
></div>

Bouton grand

<div 
  data-button="primary"
  data-text="Bouton grand"
  data-size="large"
  data-url="#"
></div>

Bouton désactivé

<div 
  data-button="primary"
  data-text="Bouton désactivé"
  data-disabled="true"
></div>

Utilisation programmatique

Les boutons peuvent également être créés et ajoutés par programmation via JavaScript.

Ajout programmatique

// Ajouter un bouton via JavaScript
import { addButtonTo } from './utils/initComponents.js';

addButtonTo('#button-container-js', {
  text: 'Contacter maintenant',
  type: 'accent',
  size: 'default',
  url: '/contact.html',
  icon: buttonIcons.mail,
  iconPosition: 'left'
});

Création directe

// Créer un bouton directement
import { createButton } from '../components/common/Button.js';

const buttonHTML = createButton({
  text: 'Télécharger PDF',
  type: 'outline',
  icon: buttonIcons.download
});

document.getElementById('button-direct-js').innerHTML = buttonHTML;
