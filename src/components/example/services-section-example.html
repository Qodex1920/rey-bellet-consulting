Exemples d'utilisation du composant ServicesSection

Cette page présente différentes façons d'utiliser le composant ServicesSection.js pour créer des sections de services interactives dans vos pages web.

Utilisation basique

L'exemple le plus simple d'intégration du composant ServicesSection avec les options par défaut.

import { createServicesSection } from '../sections/ServicesSection.js'; // Insérer la section avec les options par défaut const servicesHTML = createServicesSection(); document.getElementById('services-container').innerHTML = servicesHTML; // N'oubliez pas d'initialiser Alpine.js après l'injection import { initAlpine } from '../../utils/componentInjector.js'; initAlpine();

Personnalisation des options

Comment personnaliser la section de services avec vos propres titres, sous-titres et données de services.

import { createServicesSection } from '../sections/ServicesSection.js'; // Options personnalisées const options = { sectionId: 'services-personnalises', title: 'Services Personnalisés', subtitle: 'Découvrez notre gamme de services adaptés à vos besoins spécifiques', enterpriseServices: [ { title: 'Stratégie Digitale', description: 'Définissez votre stratégie numérique pour atteindre vos objectifs', features: ['Analyse de marché', 'Positionnement digital', 'Plan d\'action'], icon: 'strategy', ctaText: 'Demander un devis', ctaUrl: '/contact.html' }, // Ajoutez d'autres services selon vos besoins ], personalServices: [ { title: 'Site Web Vitrine', description: 'Présentez votre activité avec un site web professionnel', features: ['Design moderne', 'Optimisation mobile', 'Référencement SEO'], icon: 'website', ctaText: 'Voir nos offres', ctaUrl: '/services.html' }, // Ajoutez d'autres services selon vos besoins ] }; // Générer et insérer la section personnalisée const servicesHTML = createServicesSection(options); document.getElementById('custom-services-container').innerHTML = servicesHTML; // Initialiser Alpine.js import { initAlpine } from '../../utils/componentInjector.js'; initAlpine();

Injection via fonction dédiée

Utilisation de la fonction injectServicesSectionTo qui s'occupe à la fois de l'injection et de l'initialisation d'Alpine.js.

import { injectServicesSectionTo } from '../sections/ServicesSection.js'; // Options personnalisées const options = { title: 'Services par Injection', subtitle: 'Cette section a été injectée via la fonction dédiée' }; // Injecter la section et initialiser Alpine.js en une seule étape injectServicesSectionTo('#injected-services-container', options) .then(() => console.log('Section de services injectée avec succès')) .catch(error => console.error('Erreur lors de l\'injection:', error));

Utilisation programmatique avec bouton

Chargement dynamique de la section de services lors du clic sur un bouton.

document.getElementById('load-services-btn').addEventListener('click', async () => { // Importer dynamiquement le module const { injectServicesSectionTo } = await import('../sections/ServicesSection.js'); // Options personnalisées const options = { title: 'Services Chargés Dynamiquement', subtitle: 'Cette section a été chargée après le clic sur un bouton' }; // Injecter la section await injectServicesSectionTo('#dynamic-services-container', options); // Désactiver le bouton après chargement document.getElementById('load-services-btn').disabled = true; document.getElementById('load-services-btn').textContent = 'Section chargée'; });
